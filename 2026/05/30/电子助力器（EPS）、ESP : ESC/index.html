电子助力器（EPS）、ESP / ESC
2026-05-30 | | car

电子助力器和车身稳定系统（ESP/ESC）是汽车上两类完全不同的电子控制系统

1. 电子助力器（EPS：Electric Power Steering）

电子助力器通常指电子助力转向系统（EPS）。

它的作用是：

  • 帮助驾驶员转动方向盘
  • 降低转向所需的力气
  • 提高低速时的操控舒适性
  • 高速时减小助力，提高行驶稳定性

工作原理

传统汽车使用液压泵提供转向助力，而EPS使用：

  • 转矩传感器（检测驾驶员施加的转向力）
  • 车速传感器
  • 电机
  • ECU控制器

当驾驶员转动方向盘时：

  1. 转矩传感器检测转向意图；
  2. ECU计算需要多少助力；
  3. 电机输出相应转矩帮助转向。

优点

  • 节能（没有液压泵持续消耗发动机功率）
  • 结构简单
  • 维护成本低
  • 易于实现自动驾驶和车道保持功能

2. ESP / ESC（Electronic Stability Program / Electronic Stability Control）

ESP和ESC本质上是同一种技术。

不同厂家叫法不同：

  • ESP：最早由 Bosch 提出
  • ESC：国际通用名称
  • 其他品牌名称：
    • VSC（丰田）
    • VDC（日产）
    • DSC（宝马）
    • PSM（保时捷）

作用

防止车辆：

  • 甩尾（转向过度）
  • 推头（转向不足）
  • 湿滑路面失控
  • 紧急避让时侧滑

可以理解为：

ESP是汽车的“电子平衡教练”。

ESP如何工作

ESP会实时监测：

  • 方向盘转角
  • 横摆角速度（Yaw Rate）
  • 侧向加速度
  • 车轮转速

通过比较：

  • 驾驶员想去的方向
  • 车辆实际运动方向

判断车辆是否失稳。

情况1：转向不足（推头）

驾驶员想左转：

前轮抓地力不足。

ESP会：

  • 制动内侧后轮
  • 调整发动机输出

帮助车辆转进去。

情况2：转向过度（甩尾）

驾驶员左转：

车尾开始向外甩。

ESP会：

  • 制动外侧前轮
  • 降低发动机扭矩

把车身拉回稳定状态。

ESP与ABS、TCS的关系

ESP是在ABS和牵引力控制基础上发展出来的。

ABS（防抱死制动系统）

防止刹车时车轮锁死。

TCS/ASR（牵引力控制系统）

防止驱动轮打滑。

ESP/ESC

利用：

  • ABS液压模块
  • TCS控制功能
  • 更多传感器

实现整车姿态控制。

关系可以理解为：

简单总结

系统 全称 主要作用
EPS Electric Power Steering 给方向盘提供转向助力
ABS Anti-lock Braking System 防止制动抱死
TCS/ASR Traction Control System 防止驱动轮空转
ESP/ESC Electronic Stability Program / Control 防止车辆侧滑和失稳

一句话概括：

  • EPS解决“方向盘转起来费不费劲”的问题。
  • ESP/ESC解决“车子会不会失控甩出去”的问题。
